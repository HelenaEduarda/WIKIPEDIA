# Wikipedia para Calouros
### Projeto desenvolvido por Eduarda Helena 

   A Wikipedia para calouros é um projeto pensado e desenvolvido para aula de Fundamentos da Engenharia de Software. Consiste em desenvolver arquivos .md para leitura e compreensão de novos alunos sobre o funcionamento das primeiras matérias e semanas de aula! 
   O objetivo é conduzir os estudantes, calouros, para uma melhor experiência nas dependências da faculdade. Assim, talvez, mais alunos de identifiquem ou criem melhores expectativas sobre o curso e o primeiro período.
