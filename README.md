# WIKIPEDIA PARA CALOUROS
### Esse projeto consiste, de maneira prática, explicar como o mini mundo do curso de engenharia de software da Campo Real funciona.

No primeiro período você terá aulas das matérias de Lógica de Programação, Criatividade e Inovação, Fundamentos da Engenharia de Software e Programação 1. Abaixo vou discorrer um pouquinho sobre cada uma delas.

## Lógica de Programação
Particularmente, minha matéria favorita. Quem aplica essa matéria é a professora Adriana, ou Loli, ela é muito querida e tem uma ótima didática, o que até então se dá ao fato dela ser professora de fundamental 1 e ensino médio durante o dia. Se você esteve perdido em matemática no ensino médio ou por azar teve péssimos professores, agora é hora de correr atrás do prejuizo. A matéria não consiste só em matemática, é muito mais lógica, mas uma coisa é ligada na outra, mas pelo lado positivo, também é momento de fazer amigos então encontre alguém que possa te ajudar!

Como dica: Nesse momento é aonde você pode se socorrer nos seus colégas, sempre tem algum que vai melhor na matéria e vai saber te ajudar!
